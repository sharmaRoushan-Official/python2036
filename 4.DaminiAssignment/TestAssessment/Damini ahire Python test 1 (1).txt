18. What is packing variable in tuple? Explain using example? 
ANS
Packing means storing multiple values in one tuple.
Example:
t = 1,2,3
print(t)
Output:
(1,2,3)


19. What is unpacking variables in tuple? Explain using example?
ANS
Unpacking means storing tuple values into separate variables.
Example:
t = (1,2,3)
a,b,c = t
print(a)
print(b)
print(c)
